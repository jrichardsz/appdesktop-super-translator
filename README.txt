super-translator
================

La aplicacion lista para ser usada esta en la carpeta "target".

Modo automatico:

Para usar la aplicacion en este modo, se debe presionar el boton "start".
Luego cualquier texto que se copie (control+c) se traducira automaticamente!!

No olvidar:  Para lograr la traduccion automatica, la aplicacion esta leyendo y limpiando constantemente el 
clipboard del sistema. Por ello si se presioan el boton start, cualquier texto que se copie ya no se podra pegar(control+v).
Por ello cuando la aplicacion no se use, PRESIONAR EL BOTON STOP.

Modo Asincrono

Para usar la aplicacion en este modo, se debe presionar el boton "now". Tambien se logra esto presionando enter, cuando se este sobre el input español.
Luego cualquier texto que se ingrese en el input español, se traducira!!
